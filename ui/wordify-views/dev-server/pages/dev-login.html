Dev Login

The OAuth login flow only works on the real server. Follow these steps to transfer your session:

  1. Log in on the real server (if you haven't already)
  2. On the real server, open DevTools (F12) and go to the Network tab
  3. Refresh the page on the real server to capture a request
  4. Click any request and find the Cookie request header
  5. Copy just the value after _SESSION= (everything up to the next ; or end of string)
  6. Paste it below and click "Set Session"
The Cookie header looks like: _SESSION=abc123xyz...
Copy only the abc123xyz... part.
